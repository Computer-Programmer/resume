NSF Computer Programming Class

The NSF Computer Programming Class is a great way for children to start their journey as programmers. My name is Jatin Punjabi and I was a student in the class. I had a great time. It introduced me to basic code that I can use and become a developer soon! My teacher was Ms. Prianka Subramanyam. She is a great teacher! When I struggled, I could always ask her for help.


Here are some projects I did in Class or for Homework:
